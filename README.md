# my_test (Tkinter)

Простой интерактивный тест на Python с графическим интерфейсом на Tkinter. Вопросы хранятся в JSON, порядок вопросов перемешивается, результат подсчитывается автоматически.

## Функционал

- Отображение вопросов и вариантов ответов (Radiobutton).
- Проверка выбранного ответа и подсчёт правильных ответов.
- Перемешивание порядка вопросов при старте.
- Финальный экран с результатом (количество правильных ответов).
- Обработка крайнего случая: корректная остановка теста по лимиту вопросов без ошибок индексации.

## Стек технологий

- Python 3
- Tkinter (GUI)
- JSON (работа с данными)
- Стандартная библиотека (random, tkinter.messagebox)
